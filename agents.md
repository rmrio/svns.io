# AGENTS.md — svns.io

## О проекте

Лендинг веб-студии **svns.io**, специализирующейся на веб-разработке, внедрении AI в малый бизнес и предоставлении доступа к языковым моделям.

**Целевая аудитория:** малый бизнес, который хочет внедрить AI-технологии.

---

## Стек технологий

- **Frontend:** Vanilla HTML5 / CSS3 / JavaScript (ES6+) — без фреймворков и сборщиков
- **Backend:** Vercel Serverless Functions (Node.js)
- **AI API:** MiniMax API (модель MiniMax-M2.5)
- **Деплой:** Vercel (статика + serverless)
- **Шрифты:** Google Fonts — Syne (заголовки), Plus Jakarta Sans (основной текст), Fira Code (моноширинный)
- **Зависимости:** нет (zero-dependency подход)

## Архитектурные решения

### Zero-dependency подход
Проект намеренно не использует фреймворки, библиотеки и сборщики. Весь код — чистый HTML/CSS/JS. Это обеспечивает минимальный размер, максимальную скорость загрузки и отсутствие внешних зависимостей.

### Единственный API-эндпоинт
`/api/chat` — серверless-функция на Vercel, обрабатывающая запросы к AI. База знаний загружается из `knowledge.json` при каждом вызове. Таймаут — 30 секунд.

### Двуязычность (RU/EN)
Реализована через `data-ru` / `data-en` атрибуты в HTML. Язык переключается динамически через JavaScript и сохраняется в `localStorage`. Русский — язык по умолчанию.

### Хранение данных
Базы данных нет. `knowledge.json` хранит контекст для AI-ассистента (услуги, цены, FAQ, контакты). Пользовательские данные не сохраняются на сервере.

## Дизайн-система

### Цветовая палитра
- Фон: `#09090b` (почти чёрный)
- Акцент: `#c8ff00` (лаймовый/шартрёз)
- Текст: off-white на тёмном фоне

### Типографика
- Заголовки: **Syne** (геометрический, жирный)
- Основной текст: **Plus Jakarta Sans** (современный, чистый)
- Моноширинный: **Fira Code** (для тегов и кода)

### Визуальные эффекты
- SVG noise-оверлей для текстуры
- Тонкие grid-линии в hero-секции
- Мягное свечение акцентного цвета
- Scroll-анимации через IntersectionObserver
- Custom easing для переходов

### Принципы дизайна
- Тёмная тема (tech noir + швейцарская типографика)
- Карточная раскладка с тонкими бордерами
- Щедрые отступы (whitespace как элемент дизайна)
- Избегать типичной «AI-эстетики» — дизайн должен быть запоминающимся и самобытным

## Правила разработки

### JavaScript
- IIFE для инкапсуляции
- Event delegation
- IntersectionObserver для анимаций появления
- Async/await для API-вызовов
- Никаких внешних библиотек

### CSS
- CSS Custom Properties для дизайн-токенов
- BEM-подобное именование (`component__element--modifier`)
- Mobile-first подход (от 320px)
- Плавные transition с custom easing

### HTML
- Семантическая разметка (HTML5)
- ARIA-атрибуты для доступности
- Data-атрибуты для интернационализации
- Inline SVG для иконок

## Чат-виджет

- Плавающая кнопка в правом нижнем углу
- Интеграция с `/api/chat` (MiniMax API)
- Контекст из `knowledge.json` передаётся в system prompt
- Двуязычные ответы
- Индикатор набора текста
- При необходимости перенаправляет в Telegram: `@svns_io`

### Парсинг ответа MiniMax API
Ответ содержит массив `content` с блоками `thinking` и `text`. Необходимо находить элемент с `type: "text"` (не предполагать, что он всегда первый).
